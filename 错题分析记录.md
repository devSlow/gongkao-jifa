# 错题分析记录

## 使用方法
把做错的行测题发给我（拍照、截图、或者直接打字都行），我会：
1. 判断这道题考的是什么知识点
2. 分析你可能错在哪里（计算？思路？还是根本不知道怎么做？）
3. 用大白话讲清楚这道题的核心思想
4. 给你一道同类型的变种题检验你是否真的会了

## 和每日学习记录怎么配合

1. 每天做题时，不用把所有错题都抄到这里。
2. 当天只挑 1-3 道最有代表性的错题，先记到当天的每日学习记录里。
3. 如果某个错误连续出现 2 次以上，再把它正式记到这里，作为长期薄弱点。
4. 也就是说：
   - `每日学习记录/` 负责记录“今天发生了什么”
   - `错题分析记录.md` 负责沉淀“哪些问题反复出现、值得长期追踪”
