# 行测标准化操作流程（SOP）
# 适用对象：有基础，但做题时经常没思路、容易卡住的考生
# 核心原则：先判题型，再套流程；不要空想，要落到纸上

========================================
# 一、总原则：所有模块都按同一套逻辑做

## 通用四步

【第1步：先判题型】
不要一上来就做，先判断这题属于哪个子模块。

【第2步：先写关键信息】
把题目最关键的数字、关键词、结论、问法写出来。

【第3步：按模块模板走】
每个模块都有固定模板，按顺序填，不靠灵感。

【第4步：超时立刻止损】
行测不是主观题，卡住就先放，先保全局得分。

## 你做题时必须统一执行的规则

1. 不允许脑子里空转，必须在纸上留下步骤
2. 不允许一道题卡太久
3. 不允许做完题却不复盘错因
4. 不允许同一种错法重复出现还不记方法笔记

========================================
# 二、模块入口：先判自己现在该用哪张模板

## 1. 数量关系

遇到以下特征，直接用 `数量关系草稿纸模板.md`
- 有数字关系
- 问多少、几天、几种、利润、速度、人数
- 需要列式、代入、判断模型

核心目标：把“完全没思路”变成“先抄数字、标含义、判模型”

## 2. 资料分析

遇到以下特征，直接用 `资料分析草稿纸模板.md`
- 有表格、图表、统计材料
- 问增长率、增长量、比重、倍数、平均数
- 需要找数据、套公式、估算

核心目标：把“不会找数”变成“先定位，再判题型，再速算”

## 3. 言语理解

遇到以下特征，直接用 `言语理解草稿纸模板.md`
- 逻辑填空
- 主旨概括、意图判断、细节判断
- 语句排序、标题选择、下文推断

核心目标：把“凭感觉选”变成“先抓信号词，再找中心，再排错误项”

## 4. 判断推理

遇到以下特征，直接用 `判断推理草稿纸模板.md`
- 定义判断
- 类比推理
- 图形推理
- 逻辑判断、加强削弱、翻译推理

核心目标：把“好像都像”变成“逐条证明哪个不符合”

## 5. 政治理论

遇到以下特征，直接用 `政治理论草稿纸模板.md`
- 政治理论单独板块题
- 习近平新时代中国特色社会主义思想
- 二十大、时政热词、党建、制度、发展战略

核心目标：把“背过但不会做题”变成“先定知识领域，再用规范表述排除”

## 6. 常识判断

遇到以下特征，直接用 `常识判断草稿纸模板.md`
- 法律、科技、历史、人文、地理、经济、生活常识
- 纯记忆型或辨识型题目

核心目标：把“纠结半天”变成“快排、快选、快走”

# 三、各模块标准流程总表

## 数量关系标准流程

1. 抄数字
2. 标含义
3. 说清楚问什么
4. 抓关键词
5. 判模型
6. 写核心公式
7. 列关系
8. 不会就代入选项
9. 2分钟还不会就猜
10. 复盘到底错在模型、列式还是计算

## 资料分析标准流程

1. 先看问法
2. 回材料定位数据
3. 判断题型
4. 写公式
5. 判断是否可以估算
6. 速算或比较大小
7. 综合分析先做最好排的项
8. 90秒超时就止损
9. 复盘是定位错、公式错还是估算错

## 言语理解标准流程

1. 先判题型
2. 圈信号词
3. 找中心句或搭配点
4. 排掉偏题、偷换、过度引申、搭配错误项
5. 二选一时选更贴中心、更少引申的
6. 60秒超时就按题型规则快选
7. 复盘是中心找错还是被语感带跑

## 判断推理标准流程

1. 先判子题型
2. 抓关键词、关系、结论或图形变化
3. 按对应规则逐项对照
4. 不是找“最像”，而是找“谁不符合”
5. 图形30秒没规律就放
6. 逻辑题先找结论和论据
7. 复盘是题型判错、关系判错还是关键词漏看

## 政治理论标准流程

1. 先看属于哪块理论体系
2. 圈关键词：根本、核心、首要、目标、原则等
3. 先排表述不规范、主体错误、层级错误项
4. 二选一时优先选更规范、更完整、更像教材原话的
5. 30秒内必须出手
6. 复盘要落实到具体知识点

## 常识判断标准流程

1. 先判领域
2. 抓最明显的判断线索
3. 快排明显错误项
4. 两项纠结时选更稳妥、更不绝对的
5. 5到10秒没把握就走
6. 复盘只记最值得补的一条知识

# 四、考场时间分配 SOP

## 推荐顺序

1. 政治理论
2. 常识判断
3. 言语理解
4. 判断推理
5. 资料分析
6. 数量关系

## 推荐时间

- 政治理论：8分钟
- 常识判断：8分钟
- 言语理解：25分钟
- 判断推理：35分钟
- 资料分析：30分钟
- 数量关系：14分钟

## 超时红线

- 政治理论：30秒一题
- 常识判断：10到30秒一题
- 言语理解：60秒一题
- 判断推理：60秒一题
- 资料分析：90秒一题
- 数量关系：2分钟一题上限

只要超过红线，就不要恋战。

# 五、统一错题复盘 SOP

无论哪个模块，错题都只复盘三件事：

## 1. 这题属于什么题型
必须先归类，不能只记答案。

## 2. 我错在哪一步
从下面选一个主因：
- 没判对题型
- 关键信息漏了
- 套错流程了
- 计算/比较失误
- 时间分配失控
- 靠感觉选错

## 3. 只写一句方法笔记
格式固定：
“下次看到________，先________。”

例子：
- 下次看到“至少一个”，先想反面。
- 下次看到资料题问增长率，先分清现期和基期。
- 下次看到主旨题转折词，先看转折后。

# 六、你的训练方式也要标准化

## 平时练习怎么做

1. 做题前先准备对应模板
2. 每次只练一个模块或一个小题型
3. 不求一口气做很多，先把流程走顺
4. 错题不抄全文，只记题号、错因、方法笔记
5. 三天后重做一次同类题，检查是否真会

## 你和我配合时怎么做

你把题发给我以后，我们固定按这个顺序处理：
1. 我先判断题型
2. 我告诉你该用哪张模板
3. 我带你一格一格填
4. 我指出你到底卡在哪一步
5. 我给你一句方法笔记
6. 我再给你一题变式检查是否真正学会

========================================
# 七、最后一句话

**你现在最缺的不是“再听更多课”，而是把每个模块都变成可执行的固定动作。**

以后你不要再问自己“这题我有没有思路”，你只问自己：

**“这题该拿哪张模板，先填哪一步？”**
