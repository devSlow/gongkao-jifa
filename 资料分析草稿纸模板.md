### 第4步：写对应公式

公式：______________________________________________

常用提醒：
- 增长率 = (现期 - 基期) / 基期
- 增长量 = 现期 - 基期
- 基期 = 现期 / (1 + 增长率)
- 比重 = 部分 / 整体
- 平均数 = 总量 / 份数
- 倍数 = A / B
