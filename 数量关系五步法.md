# 数量关系五步法

## 核心理念
你不需要"灵光一闪"想到解法。你需要的是一套**机械化的步骤**，拿到任何题都照着做。
就像做菜不需要创意，照着菜谱一步步来就行。

## 五步法

### 第一步：把题目里所有数字抄出来
不管有没有用，先把所有数字列在草稿纸上。
这样做的好处：你不用在脑子里同时装着题目和计算，纸上有数字，脑子就空出来了。

### 第二步：每个数字旁边写上"这是什么"
用最简单的话标注每个数字的含义。
比如：150人/分钟 → 出口速度。1040人 → 总人数。

### 第三步：用一句话说出"题目要我求什么"
不要想怎么求，先搞清楚"求的是啥"。
问自己：最后的答案会是一个数字？一个时间？一个价格？

### 第四步：找关系——这些数字之间能不能用加减乘除连起来？
这是最难的一步，但有一个笨办法：
- 如果题目出现了"共""合计""总共" → 试试加法
- 如果题目出现了"每""单价""速度""效率" → 试试乘法（单价x数量=总价）
- 如果题目出现了"比...多/少""差" → 试试减法
- 如果题目出现了"平均分成""每份" → 试试除法
- 如果题目出现了"等于""是""相当于" → 那就是等号，两边写出来

### 第五步：列式，算，看选项
列出来之后能算就算，算不出来就看选项能不能代入排除。

## 重要提醒
- 第一步和第二步是最关键的，做到了这两步你就已经把题目"翻译"成了数学语言
- 第四步不确定的时候，就随便试一种运算，算出来的数不对再换
- 不要想一步到位，笨办法就是好办法
