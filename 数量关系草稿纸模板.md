# 数量关系草稿纸模板

> 用法：拿到任何数量关系题，不要直接想答案。先把这张模板按顺序填一遍。
> 目标：把“脑子空白”变成“纸上有步骤”。

### 第5步：先判模型
这道题最像哪一类？

- [ ] 工程问题

- [ ] 方程 / 不定方程
- [ ] 数字特性
- [ ] 容斥问题
- [ ] 概率问题
- [ ] 其他 / 没看出来

如果没看出来，就直接走“代入选项法”。

---

### 第6步：写核心公式
把这一类题最核心的公式先写出来，不急着代数字

公式：______________________________________________

如果是工程：总量 = 效率 × 时间
如果是行程：路程 = 速度 × 时间
如果是利润：利润 = 售价 - 成本
如果是概率：概率 = 满足条件 / 总情况数

## 最重要的一句话

**你不会做题的时候，不要想答案；先把模板填完。**

填完模板以后，就算还没做出来，至少你已经知道：
1. 题目给了什么数字
2. 每个数字是什么意思
3. 它最像什么模型
4. 能不能代入选项试出来
