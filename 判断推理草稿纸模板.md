# 判断推理草稿纸模板

> 用法：拿到任何判断题，不要一眼凭印象选，先判子题型，再按对应步骤排。
> 目标：把“好像都对”变成“明确知道哪一条不符合”。

### 第3步：按题型走标准流程

定义判断：
- [ ] 只按定义，不靠常识
- [ ] 任一关键词不符就排除

类比推理：
- [ ] 先判关系类型
- [ ] 再看顺序是否一致
- [ ] 再看强弱是否一致

图形推理：
- [ ] 先位置
- [ ] 再数量
- [ ] 再样式
- [ ] 最后属性

加强/削弱：
- [ ] 先找结论
- [ ] 再找论据
- [ ] 看选项是否搭桥/拆桥

翻译推理：
- [ ] 写成 A → B
- [ ] 只认逆否，不乱逆乱否

## 最重要的一句话

**判断推理不是想“哪个像”，而是要证明“哪个不符合”。**
