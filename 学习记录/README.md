# 学习记录

这个目录可以直接作为 Obsidian 的一个仓库，或者作为你现有仓库里的一个学习记录区。

## 目录说明

- `模板/`：固定模板
- `记录/`：你真正每天/每周要写的文件

## 最简使用方法

1. 先去 `模板/` 找对应模板
2. 在 `记录/` 里新建当天或当周文件
2. 按模板填写，控制在 5-10 分钟内
3. 重点写：今天卡在哪、今天想通了什么、明天第一件事做什么

## 推荐节奏

- 每天：写一篇 `每日记录`
- 每周：写一篇 `每周复盘`

周复盘不是重复写日记，而是把这一周反复出现的问题和真正有效的方法拎出来。
对你现在的备考来说，周复盘比“多看一节课”更值钱，因为它能帮你看见：

- 这一周到底学了什么
- 哪些题型只是看过，但还不会
- 哪些错法在重复出现
- 下周最该补的到底是什么

## 记什么最有用

不要写流水账，比如“今天看了两小时课”。
尽量写成下面这种：

- 今天做了哪一类题
- 卡在什么地方
- 我今天真正明白的一句话是什么
- 明天先补哪一个点

## 建议文件名

- `2026-06-19-每日.md`
- `2026-W25-周复盘.md`
