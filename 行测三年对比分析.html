考两次了，到底差在哪

两次考试成绩：行测 47 -> 56，申论约50。残疾人专岗进面线140+，竞争对手大部分也在60多但同样上不了岸。 你目前总分106，需要再提34分。目标：行测70 + 申论70 = 140，这是分数线要求，不是"超过谁"的问题。

申论 -- 最大失分点
50分 -> 目标70分
现状
  • 看了飞扬等老师的网课，但自己动笔写得很少
  • 申论是主观题，光看课等于没学。就像看别人游泳不下水，永远学不会
  • 竞争对手申论70分，你50分，20分差距直接决定能不能进面
你的核心问题
  • 归纳概括：要点找不全、表述不精练，答案写得像在"叙述"而不是"概括"
  • 综合分析：逻辑框架不清楚，分析没有层次感
  • 公文写作：格式可能知道，但内容组织能力不够
  • 大作文：立意可能偏、论证空洞、缺少论据储备
怎么从50冲到70
  1. 每周写2套申论真题（严格限时），写完对照参考答案逐句比对
  2. 前两周只练"归纳概括"和"提出对策"，这两种题型最容易拿分
  3. 积累规范表述：把参考答案里的"标准表述"抄到一个本子上，反复看
  4. 大作文先练审题立意（只写提纲），8月再开始写全文
  5. 找人批改或用AI批改（把答案发给我，我帮你看），自己看不出自己的问题
  6. 申论70分不是靠天赋，是靠"写够量"。目标：考前写完至少15套完整真题
数量关系 -- 看了课但不会做题
~4题/15题
现状
  • 看了好几个老师的课（花生十三、齐麟等），但题目做得很少
  • 课上听懂了，课后不会做，这是"被动学习"的典型症状
  • 排列组合、概率等你之前说"根本不会"
为什么会这样
  • 数量关系的本质是数学应用题，光看解题过程不行，必须自己算
  • 换老师没用，每个老师讲的方法大同小异，缺的是你自己的练习量
  • 考场上需要的是"条件反射"级别的熟练度，不是"理解"
具体怎么练
  1. 不要再换老师了，就用一个老师的讲义（推荐花生十三）
  2. 每天10题数量关系，必须动笔算，不能只看答案
  3. 先只攻克4个高频题型：方程/不定方程、数字特性、工程问题、利润问题
  4. 做错的题隔3天重做一遍，直到能独立算对
  5. 排列组合只学最基础的两类（分类加法、分步乘法），考场遇到复杂的直接猜
资料分析 -- 公式知道但算得慢
~10题/20题
现状
  • 2025/2026年资料分析增加到20题，分值占比上升
  • 基本公式可能知道，但速算不过关，算一道题花太久
  • 综合分析判断题（5题）最容易丢分，因为需要逐项验证
具体怎么练
  1. 先花3天专门练"截位直除"和"特征数字法"，这是资料分析的灵魂
  2. 每天做1组（5题），限时8分钟。超时说明速算不行
  3. 综合分析题先看选项再回材料找数据，不要从头算到尾
  4. 目标：8月底前正确率到85%（17题/20题）

  3. 每天早上花20分钟背诵，晚上花10分钟回顾
  4. 政治理论是"背了就有分"的模块，15题争取对11-12题

言语 ~17/25   判断 ~24/35

  • 言语理解的逻辑填空靠语感而不是方法，波动大
  • 判断推理的定义判断和类比推理是送分题，你没有做到90%正确率
具体怎么练
  1. 言语：每天20题逻辑填空，做完把错题的成语/实词辨析抄到本子上
  2. 判断：定义判断和类比推理单独刷，目标正确率90%
  3. 图形推理和逻辑判断跟花生十三的方法论，不要靠感觉
  4. 这两个模块不需要看新课，就是刷题+总结错题
常识判断 -- 没看过课，但不用专门补
~7题/15题
  • 常识判断覆盖面极广（法律、科技、历史、地理），专门复习的投入产出比很低

先校准目标：行测80分不现实，但上岸完全可行

行测80分意味着什么：125题只能错15-20题。你目前大概错了55题。4个月内从错55题减到错15题，坦率说，我没有见过这样的案例。行测72-75分在安徽已经是顶尖水平。

但你的目标不是"考80分"，你的目标是"考140上岸"。行测70+申论70+=140，两个科目都得冲70+，因为竞争对手60多也上不了岸，140是硬门槛。

错题怎么处理：不要抄题，要抄"方法"

不要做的事：把纸质书的错题打字打出来一题题搜解析
  • 抄一道题+搜解析+看懂，至少5分钟。100道错题就是8小时
  • 抄完你记住的是"这道题选C"，不是"这类题怎么解"
  • 这是典型的"用体力代替思考"，看起来很努力，实际效率极低
正确做法：错题本只记"方法笔记"，不抄原题
具体操作步骤
  1. 在纸质书上直接用红笔标记错题题号，不要抄出来
  2. 看完解析后，在错题本上只写一行："XX题型 + 解题方法"
    例如："增长率比较：分子大分母小则大，分子小分母大则小，交叉看趋势"
    例如："逻辑填空'南辕北辙'强调行动和目的相反，'背道而驰'强调方向相反"
  3. 同一类错题归到一起，你会发现80%的错题都是重复犯同样的错误
  4. 每隔3天把标记的错题重做一遍（不是看答案，是重新做）
  5. 连续做对2次的错题可以划掉，不再管它
错题本的本质：不是记录"我错了什么"，而是记录"我不会什么方法"。一本好的错题本应该只有几页纸，写满了你曾经不会的解题方法和易混淆知识点。

每天到底做什么：一张清单

不用想太多，每天照着这个顺序做就行，大约4-5小时

第一件事（最重要）：申论动笔写 -- 60分钟
  • 打开粉笔APP，找一套申论真题
  • 只做前2道小题（归纳概括/提出对策），严格限时各25分钟
  • 写完立刻看参考答案，逐句比对：你漏了哪些要点？你的表述和标准表述差在哪？
  • 把参考答案里的"规范表述"抄到本子上（只抄你没想到的那些）
  • 目标：考前写完至少15套完整申论真题，这是从50到70的唯一路径
第二件事：资料分析限时练 -- 40分钟
  • 粉笔APP刷5道资料分析，手机计时8分钟
  • 超时的题做完后看解析，重点看速算方法
  • 错题在书上标记，3天后重做
第三件事：判断推理+言语理解刷题 -- 60分钟
  • 定义判断10题 + 类比推理10题 + 逻辑填空20题
  • 做完对答案，错题标记+记方法笔记
第四件事：政治理论背诵 -- 20分钟
  • B站或粉笔找政治理论专题，每天背1个专题的核心表述
第五件事：数量关系做题 -- 20分钟
  • 只做方程题+数字特性题，每天5题，必须动笔算
  • 不要看新的课了，就做题。不会的题看解析，记方法
关键原则：从今天开始，不再看任何新课。你的问题不是"学得不够"，而是"练得不够"。每天的4-5小时全部用来做题+对答案+记方法，不要再花一分钟看视频课。

什么是"立足于基础"

数学家葛军一直强调"立足于基础"。很多人以为基础就是"背公式"，其实不是。基础 = 理解公式背后的等量关系。

你目前的状态：改改数字会做，换个场景就不会。这说明你记住了"解题模板"，但不理解"为什么要这样列式"。 下面每个题型，我都从"一句话说清本质"开始，帮你建立真正的理解。

一、工程问题

行测数量关系每年必考1-2题，属于中等难度，学会了稳拿分
核心思想（整个工程问题只靠这一句话）
总量 = 效率 x 时间
不管题目怎么变，都是在问"谁、以多快的速度、干了多久、干了多少"

为什么是这个公式

1
大白话说：你搬100块砖，每小时搬10块，要搬10小时。100 = 10 x 10。就这么简单。
2
公式变形：知道总量和效率，可以算时间（时间 = 总量 / 效率）。知道总量和时间，可以算效率（效率 = 总量 / 时间）。三个量，知道任意两个，第三个就能算。
3
多人合作：甲的效率 + 乙的效率 = 两人合作的效率。为什么能直接加？因为效率就是"单位时间的工作量"，两个人同时干，单位时间干的量就是各自干的量之和。
基础例题
一项工程，甲单独做需要12天，乙单独做需要18天。两人合作几天完成？
1
问自己：这道题在问什么？
问的是"合作几天完成"，即求时间。已知两个人各自的效率（通过"单独做几天"间接给出），求合作时间。
2
怎么处理"甲单独做12天"？
"单独做12天"= 效率是总量的1/12。为什么？因为总量 / 12天 = 每天做的量 = 效率。
同理，乙的效率 = 1/18。
3
合作效率 = 1/12 + 1/18 = 5/36
合作时间 = 总量(1) / 合作效率(5/36) = 36/5 = 7.2天
4
回头看：我做了什么？
从头到尾只用了一个公式：总量 = 效率 x 时间。"单独做几天"倒推出效率，合作效率相加，再除总量得时间。

变种：同一个思想，换了件衣服

变种1：注水/排水问题
一个水池，进水管每小时注水3吨，排水管每小时排水1吨。水池容量12吨，多久注满？
本质没变：净效率 = 3 - 1 = 2吨/小时，时间 = 12 / 2 = 6小时。
只是"效率"从"工作效率"变成了"净注水速度"，公式还是总量 = 效率 x 时间。
变种2：分段合作问题
甲先单独做3天，然后甲乙合作，还需几天完成？
本质没变：先算甲3天做了多少（3 x 甲的效率），剩余量 = 总量 - 已做量，然后用合作效率算剩余时间。
只是把"一步算完"拆成了"两步"，公式还是同一个。
变种3：效率变化问题
甲前5天效率为2，之后效率降低为1，总量30，共需几天？
本质没变：前5天做了 5 x 2 = 10，剩余 30 - 10 = 20，以效率1做还需 20天，共 25天。
分段处理，每段都用同一个公式。
常见坑
坑1："甲单独做12天"不是说甲做了12天，是说甲的效率 = 1/12。别把"做完需要几天"和"实际做了几天"搞混。
坑2：合作效率可以直接加，但"合作时间"不能直接加。别算出甲要12天、乙要18天，就以为合作要30天。
坑3：题目问"还需几天"时，记得减去已经做的天数。

二、排列组合

行测每年考1-2题，难度偏高。掌握基础思想够用，不必追求难题
核心思想（不是背公式，是理解"选"和"排"的区别）
选 = C（不管顺序）    排 = A（管顺序）
所有排列组合题，本质就是在问一个问题："顺序重不重要？"

为什么要除以m!（理解C的来历）

1
先理解A（排列）：从5个人里选2个人排队，有多少种排法？
第1个位置有5种选择，第2个位置有4种选择，所以 5 x 4 = 20种。
这就是 A(5,2) = 5 x 4 = 20。管顺序，先选谁后选谁不一样。
2
再理解C（组合）：从5个人里选2个人组队（不排队），有多少种选法？
A和B组队、B和A组队，是同一支队伍。所以每一对人都被A(5,2)重复计算了2次。
真正的选法 = A(5,2) / 2! = 20 / 2 = 10种。
除以m!就是去掉重复计算的部分。
3
一句话总结：选人不管顺序用C，安排位置/排队用A。拿到题目先问自己"换一下顺序结果变不变"，变了用A，不变用C。
基础例题：怎么判断用A还是C
题1：从7名志愿者中选3人分别去A、B、C三个岗位，有多少种安排方式？
题2：从7名志愿者中选3人参加活动，有多少种选法？
1
题1：问自己"换顺序结果变不变？"
甲去A岗、乙去B岗 和 甲去B岗、乙去A岗，是不同的安排。顺序变了结果变了，用A。
A(7,3) = 7 x 6 x 5 = 210种
2
题2：问自己"换顺序结果变不变？"
选了甲乙丙 和 选了丙甲乙，是同一组人。顺序变了结果没变，用C。
C(7,3) = A(7,3) / 3! = 210 / 6 = 35种
3
同一个基础：A(7,3) = 210，C(7,3) = 210 / 6 = 35。
区别只有一个：除不除m!。除不除取决于"顺序重不重要"。

变种：换了件衣服但本质没变

变种1：分组问题
6个人平均分成2组，有多少种分法？
思考：先从6个里选3个 C(6,3)=20，剩下3个自动成另一组。但两组没有标签（不是A组B组），所以除以2!去重复。答案 = 20/2 = 10种。
本质：还是"选"，但多了"去重复"这一步。
变种2：至少/至多问题
从5男3女中选3人，至少1名女性，有多少种选法？
思考方法1（正面）：1女2男 + 2女1男 + 3女0男 = C(3,1)C(5,2) + C(3,2)C(5,1) + C(3,3) = 30+15+1 = 46
思考方法2（反面）：全部 - 0女 = C(8,3) - C(5,3) = 56 - 10 = 46
本质：还是选（C），只是需要分类讨论或用"总数减反面"简化计算。
变种3：捆绑/插空问题
5个人排队，甲乙必须相邻，有多少种排法？
思考：把甲乙捆绑成一个"超级人"，和其余3人共4个元素排列 A(4,4)=24，甲乙内部可以互换 2!=2，共 24 x 2 = 48种。
本质：还是排（A），只是需要先处理限制条件。
常见坑
坑1：拿到题就套公式，不做"顺序重不重要"的判断。这是最常见的错误。
坑2：分组问题忘记除以组数的阶乘（去重复）。分2组除以2!，分3组除以3!。
坑3："至少"问题正面分类太多时，一定要想到"总数减反面"，能省一半计算量。
坑4：排列组合是考试中性价比最低的题型。如果30秒内看不出思路，直接猜答案跳过，把时间留给资料分析。

三、方程与不定方程

每年必考2题，最基础的方法，必须熟练到条件反射
核心思想
找到等量关系，设未知数，列方程
数量关系里80%的题，最终都是在列方程、解方程。区别只在于"怎么找到等量关系"

怎么找等量关系

1
看题目里有没有"等于""是""共""合计"这些词。
"甲乙共花了100元" → 甲 + 乙 = 100
"甲的年龄是乙的2倍" → 甲 = 2 x 乙
"售价比成本高20%" → 售价 = 成本 x 1.2
2
不定方程：未知数比方程多，但有隐藏约束
比如"买5元和8元的文具共花了52元，每种至少买1个"，列式：5x + 8y = 52
一个方程两个未知数，看似无解。但x和y都是正整数，这就限定了范围。
试一试：y=1时 x=44/5(不行)，y=2时 x=36/5(不行)，y=4时 x=4(行!)
这就是不定方程的解法：利用"整数"这个隐藏条件逐个试。
基础例题
小王买了单价5元的笔记本和单价3元的签字笔共20支，花了76元。问笔记本买了几支？
1
找等量关系："共20支" → x + y = 20；"花了76元" → 5x + 3y = 76
2
解方程：由第1式得 y = 20 - x，代入第2式：
5x + 3(20-x) = 76 → 5x + 60 - 3x = 76 → 2x = 16 → x = 8支
变种：鸡兔同笼
鸡有2只脚，兔有4只脚，共35个头、94只脚。问鸡几只？
本质还是方程：鸡 + 兔 = 35，2x鸡 + 4x兔 = 94。和上面的笔记本签字笔完全一样。
鸡兔同笼听起来吓人，本质就是二元一次方程组。
常见坑
坑：不定方程不要硬解，用"整除特性"可以秒杀。比如 5x + 8y = 52，52和8y都是偶数，所以5x必须是偶数，x必须是偶数。直接排除一半选项。

四、数字特性（秒杀技巧）

每年考1题，学会后可以在30秒内秒杀，性价比极高
核心思想
不计算，看数字本身的特征就能排除选项
选项是具体数字，但答案必须满足某些整除、奇偶、大小关系，不满足的选项直接排除

三种最常用的特性

1
奇偶性：偶数 + 偶数 = 偶数，奇数 + 偶数 = 奇数。
如果题目说"甲乙年龄之和是偶数"，而选项里有奇数，直接排除。
2
整除性：如果"甲的钱是乙的3倍"，那甲的金额一定能被3整除。
选项里不能被3整除的，直接排除。
3
尾数法：只算个位数。比如 37 x 23，个位 = 7 x 3 = 21，尾数是1。
四个选项尾数不同的话，只算尾数就够了。
秒杀例题
某班学生人数在40-50之间，男生人数是女生的1.5倍。问全班有多少人？
A.42   B.44   C.45   D.48
1
男生 = 1.5 x 女生 = 3/2 x 女生
所以总人数 = 男生 + 女生 = 3/2 x 女生 + 女生 = 5/2 x 女生
总人数必须是5的倍数（因为女生人数 x 5/2 = 总人数，女生人数必须是偶数使结果为整数）。
2
40-50之间5的倍数只有45。
不用列方程，不用计算，直接选 C.45。
常见坑
数字特性只能用来排除选项，不能直接得到答案（除非排除到只剩一个）。所以先用特性排除，剩下的再算。

五、利润问题

每年必考1-2题，套路固定，理解公式后非常好拿分
核心思想
利润 = 售价 - 成本    利润率 = 利润 / 成本
所有利润题，不管怎么绕，都是在问"成本、售价、利润"这三个量的关系

关键公式（理解后不用背）

1
利润率 = 利润 / 成本（注意：除以成本，不是除以售价）
成本100元，卖120元，利润20元，利润率 = 20/100 = 20%。
2
打折：打8折 = 售价变成原来的80%。如果原价100，打8折后卖80。
折扣是对"售价"打折，不是对"成本"打折。
3
核心套路：设成本为x，根据利润率算出售价，再根据"打折/促销"调整售价，最后用"总收入 = 单价 x 数量"列式。
基础例题
某商品成本200元，标价为成本的150%，后来打9折出售。问利润率是多少？
1
标价 = 200 x 150% = 300元
2
实际售价 = 300 x 90% = 270元（打9折）
3
利润 = 270 - 200 = 70元
利润率 = 70 / 200 = 35%
4
回头看：三步走：算标价 → 打折算售价 → 利润/成本。所有利润题都是这个套路。
常见坑
坑1：利润率除以的是成本，不是售价。这是最常犯的错。
坑2："标价"和"售价"不一样。标价是挂的价，售价是实际卖的价（打折后的）。
坑3："亏损20%"意思是售价 = 成本 x (1-20%)，不是售价减去20元。

六、行程问题

每年考1-2题，和工程问题是"双胞胎"，公式结构完全一样
核心思想
路程 = 速度 x 时间
和工程问题的"总量 = 效率 x 时间"结构完全一样。学了工程问题，行程问题就学了一半

对比理解

工程问题行程问题
核心公式总量 = 效率 x 时间路程 = 速度 x 时间
合作效率相加相向而行时速度相加
同向--速度相减（追及）

两种经典模型

1
相遇问题：甲乙从两地相向而行。
路程之和 = 总距离。公式：(V甲 + V乙) x 时间 = 总距离。
本质：两个人同时走，每小时"靠近"的距离 = 两人速度之和。
2
追及问题：甲从后面追乙。
路程之差 = 初始距离。公式：(V甲 - V乙) x 时间 = 初始距离。
本质：每小时"缩短"的距离 = 两人速度之差。
常见坑
坑1：单位不统一。题目给"千米/小时"，但时间给的是分钟，记得换算。
坑2：相遇问题用速度之和，追及问题用速度之差。搞反了就全错。

七、容斥问题

2年考过，公式简单，画个图就清楚了
核心思想
A或B = A + B - A且B
两个集合加在一起时，重叠部分被算了两次，所以要减掉一次
基础例题
班里40人，选了数学的有25人，选了英语的有20人，两门都选的有10人。问两门都没选的有几人？
1
至少选一门的人数 = 25 + 20 - 10 = 35
为什么要减10？因为两门都选的人在"25"和"20"里各被算了一次，总共被算了2次，要减掉1次。
2
两门都没选 = 总人数 - 至少选一门 = 40 - 35 = 5人
常见坑
三个集合的容斥公式：A或B或C = A+B+C - A且B - A且C - B且C + A且B且C。记不住公式就画韦恩图。
